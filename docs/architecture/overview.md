## жҠҖжңҜж Ҳ

### Backend
- **Framework**: FastAPI
- **Language**: Python 3.11+
- **Concurrency**: asyncio
- **Key Libraries**:
  - `watchdog` - ж–Үд»¶зі»з»ҹзӣ‘жҺ§
  - `httpx` - ејӮжӯҘ HTTP е®ўжҲ·з«Ҝ
  - `pydantic` - ж•°жҚ®йӘҢиҜҒ
  - `loguru` - ж—Ҙеҝ—з®ЎзҗҶ
  - `capstone` - еҸҚжұҮзј–еј•ж“Һ

### Frontend
- **Framework**: React 18
- **Language**: TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **State**: Zustand
- **Charts**: ECharts
- **UI Components**: Radix UI

### Infrastructure
- **Container**: Docker + Docker Compose
- **Reverse Proxy**: Nginx
- **CI/CD**: GitHub Actions
- **Registry**: GitHub Container Registry

## жү©еұ•жҖ§и®ҫи®Ў

1. **жҸ’д»¶еҢ– AI еј•ж“Һ**: ж”ҜжҢҒеҲҮжҚўдёҚеҗҢ LLM жҸҗдҫӣе•Ҷ
2. **еҸҜй…ҚзҪ®и§ЈжһҗеҷЁ**: ж—Ҙеҝ—ж јејҸйҖҡиҝҮй…ҚзҪ®е®ҡд№ү
3. **ж°ҙе№іжү©еұ•**: ж— зҠ¶жҖҒи®ҫи®ЎпјҢж”ҜжҢҒеӨҡе®һдҫӢйғЁзҪІ
4. **дәӢд»¶й©ұеҠЁ**: еҹәдәҺ asyncio зҡ„ејӮжӯҘжһ¶жһ„
